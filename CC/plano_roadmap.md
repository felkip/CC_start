# Plano do Roadmap de Estudos

## Objetivo
Criar uma versão simples de roadmap que sugira caminhos de estudo com base no resultado do quiz de perfil.

## Escopo inicial (MVP)
- Exibir um roadmap básico após o usuário concluir o quiz.
- Mostrar tópicos recomendados para cada perfil:
  - Perfil de Programação
  - Perfil de Tecnologia
  - Perfil de Início
- A sugestão pode ser apresentada em forma de lista simples.

## Fluxo da funcionalidade
1. O usuário responde ao quiz.
2. O sistema identifica o perfil mais provável.
3. O roadmap correspondente é exibido na tela.
4. O usuário pode seguir os tópicos sugeridos ou refazer o quiz.

## Estrutura proposta
- Criar um módulo separado para o roadmap.
- Armazenar os tópicos recomendados por perfil.
- Exibir a sugestão no fluxo da aplicação após o quiz.

## Sugestões iniciais de conteúdo

### Para Perfil de Programação
- Fundamentos de lógica
- Variáveis e estruturas básicas
- Funções
- Estruturas de decisão e repetição
- Introdução à programação com Python

### Para Perfil de Tecnologia
- Fundamentos de computadores
- Internet e sistemas
- Ferramentas básicas de produtividade
- Introdução a bancos de dados
- Noções de segurança e uso de tecnologia

### Para Perfil de Início
- Introdução à computação
- Conceitos básicos de tecnologia
- Primeiros passos em lógica
- Pequenos exercícios práticos
- Estudo guiado com ritmo leve

## Próximos passos
1. Criar o módulo do roadmap.
2. Definir os tópicos por perfil.
3. Integrar a exibição ao resultado do quiz.
4. Melhorar a apresentação visual no futuro.

## Observação
A ideia inicial é deixar o roadmap simples, útil e fácil de entender, antes de evoluir para uma versão mais completa.
